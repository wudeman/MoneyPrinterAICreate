【项目介绍】

1、这个项目是前后端分离的，前端使用Vue3+ElementPlus，后端使用FastAPI。
2、【重要】后端在MoneyPrinterAICreate/app目录下，启动入口是MoneyPrinterAICreate/main.py，执行python main.py即可启动后端项目。
3、后端启动后，通过访问http://localhost:8080/docs查看api。
4、【重要】前端在MoneyPrinterAICreate/sites目录下，执行npm install && npm run dev即可启动前端项目。
5、【重要】所有代码的实现需要写上注释，包括函数、类、方法等（注释需要人和ai都能看得懂）。
